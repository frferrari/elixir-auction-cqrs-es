    This event is generated when a user requests an auction to be closed before its scheduled end time

    Event.AuctionClosed

    This event is used when an auction is closed, this could happen in different cases

    Event.AuctionCreated

    This event is used when an auction is created

    Event.AuctionRenewed

    This event is created when an auction has reached its end time and is automatically renewed by the system because it has no bid and it has the automatic renewal option

    Event.AuctionRestarted

    This event is created when an auction has ended and was closed and the owner of the auction decides to restart it
